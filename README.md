# LowCortisolApp

Low Cortisol на Kotlin + Jetpack Compose

## Скачать APK
Смотри Releases
## Как использовать
Назначаете время для учебы/работы
Назначаете время для отдыха
Выбираете аудиофайл при котором будет падать кортизол
Нажимаете Старт
Учитесь
По истечении таймера включается аудиофайл и кортизол начинает падать
